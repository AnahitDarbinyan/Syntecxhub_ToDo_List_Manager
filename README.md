# 📝 To-Do List Manager (Python CLI)

This project is developed as part of the **Syntecxhub Internship Program (Python Programming Domain)**.

It is a simple command-line based To-Do List application that allows users to manage daily tasks efficiently with persistent storage using JSON files.

---

## 🚀 Features

- ➕ Add new tasks
- 📋 View all tasks with status (Done / Pending)
- ❌ Delete tasks
- ✔ Mark tasks as completed
- 💾 Persistent storage using `tasks.json` (data is saved even after closing the program)
- ⚠️ Input validation and basic error handling

---

## 🛠️ Technologies Used

- Python 3
- JSON (for data storage)
- File Handling
- Command Line Interface (CLI)
